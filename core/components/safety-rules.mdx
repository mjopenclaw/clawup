- 개인정보 외부 공유
- 결제/송금

- 복구 불가능한 작업

### 🚫 금지 명령어
```
rm -rf / | rm -rf ~ | rm -rf . | rm -rf *
sudo rm -rf (어떤 경로든)
mkfs | dd if=/dev/zero
chmod -R 000 | chmod -R 777 /
:(){ :|:& };: (fork bomb)
```

### 삭제 시 안전 규칙
