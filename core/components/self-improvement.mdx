## 🧠 자가발전 시스템

### 데이터 기반 학습 루프

```
포스팅 → 성과 측정 → 패턴 분석 → 규칙 도출 → 적용 → 반복
```

### 분석 대상

#### 1. 포스트 성과 (post_analytics)
- **언제** 올리면 좋나? (시간대, 요일)
- **무슨 주제**가 반응 좋나? (OpenClaw, AI, 자동화)
- **어떤 형식**이 좋나? (텍스트, 스레드, 이미지)
- **길이**는? (짧은 vs 긴)
- **해시태그/이모지** 영향?

#### 2. 댓글 성과 (reply_analytics)
- **어떤 계정**에 댓글 달면 맞팔 잘 오나?
- **어떤 유형** 댓글이 좋아요 많이 받나?
- **질문 vs 팁 vs 경험 공유** 뭐가 효과적?

#### 3. 성장 패턴 (growth_patterns)
- 팔로워 증가/감소 추이
- 어떤 활동이 팔로워 증가에 기여?
- 병목 구간은?

### 자가 발전 규칙 (self_improvement_rules)

```sql
-- 예시
INSERT INTO self_improvement_rules (category, rule, confidence, source)
VALUES 
  ('timing', '오후 7-9시 포스팅이 engagement 2배', 0.8, '2주간 데이터'),
  ('content', 'OpenClaw 팁 포스트가 일반 포스트보다 좋아요 3배', 0.7, '10개 포스트 분석'),
  ('engagement', '질문형 댓글이 맞팔 확률 40% 높음', 0.6, '50개 댓글 분석');
```

### 성장 정체 시 대응

```
팔로워 증가 < 주간 목표의 50%
    ↓
원인 분석 (engagement 낮음? 노출 적음?)
    ↓
대응 전략:
1. 콘텐츠 개선 (패턴 분석 기반)
2. Reply Guy 강화 (더 큰 계정 타겟)
3. 광고 검토 (민제님 승인 시)
```

### 광고 전략 (필요시)

**조건**: 팔로워 성장 2주 연속 정체
**옵션**:
- X Ads: 인기 트윗 부스트
- Threads: 협업/멘션 전략
- 예산: 민제님 승인 필요

### 주간 자가 발전 리포트

매주 일요일 분석:
1. 이번 주 최고/최저 성과 포스트
2. 도출된 새 규칙
3. 다음 주 실험 계획
4. 성장 예측
