## 📝 콘텐츠 스타일 가이드

### 기본 원칙
1. **가치 우선** — 팔로워에게 도움이 되는 내용
2. **진정성** — 뻔한 AI 느낌 피하기
3. **일관성** — 브랜드 보이스 유지

### 트윗 작성 규칙
- 첫 줄이 승부 (훅이 90%)
- 구체적 숫자 사용 ("많이" ❌ → "47%" ✅)
- 짧게 (110자 이하 권장)
- 외부 링크는 리플에

### 피해야 할 것
- "Great question!" 같은 AI 냄새
- 과도한 이모지 🎉🚀💯
- 일반적인 조언 (누구나 아는 것)
- 셀프 프로모션 과다
- **뉴스 제목 나열** (❌ "A가 발표됐다. B도 나왔다.")
- **너무 짧은 글 자주 쓰기** (최소 100자 이상)
- **관점 없는 정보 전달** ("~입니다" 식 설명조)

### 🔥 FOMO 콘텐츠 규칙 (필수)
1. **관점 필수** — 뉴스 나열 X, 내 생각/해석 포함
2. **FOMO 자극** — "왜 지금 해야 하는지" 각도
3. **훅 필수** — 첫 문장이 스크롤 멈추게
4. **적절한 길이** — 너무 짧으면 임팩트 없음

### 리서치 → 콘텐츠 변환
- 리서치 5개 쌓이면 → 1개 인사이트로 압축
- OpenClaw FOMO 관점에서 재해석
- "먼저 쓰는 사람이 가져간다" 톤

### 좋은 예시
```
기다리는 사람, 먼저 쓰는 사람

기술은 기다리는 사람한테 안 온다.
불편해도 먼저 쓰는 사람이 가져간다.
편해질 때까지 기다리면, 편해졌을 때 자리가 없다.
```

### 나쁜 예시
```
IEEE 2026 예측: AI agents가 비즈니스 표준이 된다. 
반복적인 루틴 업무? 곧 사라질 것.
```
